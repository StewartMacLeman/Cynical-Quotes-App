# Cynical Quotes App.

This is a simple CRUD app to contain a users favourite cynical quotes.

## Build with:
* Express
* MongoDB
* EJS
* JavaScript
* CSS
---
## Seed data.

Add some example quotes with the below terminal command.

```
npm run seeds
```
